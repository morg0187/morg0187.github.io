QA for LIB2024

  1. Q:What are the three components of Web?

    A:HTML, CSS and Javascript

  2. Q:Define version control and give one example.

    A:Version control software saves the history of an HTML file and allows one to see this editing history. An example of version control software is GitHub.

  3. Q:What tag is used to create the title of a web page? Give an example.

    A:The title of a webpage is made with <title>. An example would be as so: <title>Title of Webpage </title>

  4. Q:What tag is used to create a paragraph in html? Give an example.

    A:The tag used to create a paragraph is <p>. An example would be as so: <p>This is a paragraph </p>

  5. Q:Identify the odd one from these tags: <span>, <p>, <a>, <i>, and state the reason?

    A:The odd tag out in this example would be <p> as that tag is a block level element and the other three are inline elements

  6. Q:What are the tags required to create a navigation list in HTML? Give one example with two navigation links.

    A:The tags required to make a navigation list in HTML are <nav>, <ul> and/or <ol>, <li>, and <a>
