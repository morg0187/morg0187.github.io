QA for LIB2024

  1. Q:What are the three components of Web?

    A:HTML, CSS and Javascript

  2. Q:Define version control and give one example.

    A:Version control software saves the history of an HTML file and allows one to see this editing history. An example of version control software is GitHub.

  3. Q:What tag is used to create the title of a web page? Give an example.

    A:The title of a webpage is made with <title>. An example would be as so: <title>Title of Webpage </title>

  4. Q:What tag is used to create a paragraph in html? Give an example.

    A:The tag used to create a paragraph is <p>. An example would be as so: <p>This is a paragraph </p>

  5. Q:Identify the odd one from these tags: <span>, <p>, <a>, <i>, and state the reason?

    A:The odd tag out in this example would be <p> as that tag is a block level element and the other three are inline elements

  6. Q:What are the tags required to create a navigation list in HTML? Give one example with two navigation links.

    A:The tags required to make a navigation list in HTML are <nav>, <ul> and/or <ol>, <li>, and <a>

    Example of navigation:

    • Algonquin College
    • Google

  7. Q:What element is used to create images? Name its attributes.

    A:The element used to create images in HTML is <img>. Its attributes are "src" which specifies the source of the image and "height" and "width" which, when combined with a number of pixels, dictates what size the image will appear when rendered.

  8. Q:Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    A:The odd tags out is a <b> since this tag is not a table related tag.

  9. Q:Give an example of color picker input.

    A:The color picker input would <input type="color">.

  10. Q:What is the use of name and value attributes? Give two examples

    A:The name attribute is used as a reference to JavaScript elements or a form element. The value attribute is used to denote the value of an input code.

  11. Q:Give an example of inline style.

    A:Inline styling is when CSS is applied to a specific tag. Inline styling will take the highest precedence over embedded styling and external styling.

    Example:

    This paragraph is styled

  12. Q:How would you link an external style sheet? Give example.

    A:To link an external style sheet, you would first create a separate folder as a sibling to your HTML document, then create a stylesheet in that folder. Then in the head of your HTML document, you would create a link tag with an "href" attribute that would link to your folder.

    Example: <link rel="stylesheet" type="text/css" href="/css/style.css">
